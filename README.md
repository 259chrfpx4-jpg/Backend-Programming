# Vacation Package Booking — Back-End API

A Spring Boot REST API serving as the back end for a full-stack booking
application (WGU course D288, Back-End Programming).

## Tech Stack
- Java, Spring Boot (Spring Data JPA, REST controllers)
- MySQL (JPA/Hibernate ORM)
- Maven

## Features
- RESTful CRUD endpoints consumed by a provided front-end client
- JPA entities with relational mappings and enforced business rules
- Server-side validation and transactional service layer
